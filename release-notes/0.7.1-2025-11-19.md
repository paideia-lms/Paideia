# Release Notes - Paideia LMS v0.7.1

**Release Date:** November 19, 2025  
**Version:** 0.7.1  
**Type**: Feature Release & Improvements  
**Impact**: Medium - Enhanced grading system, discussion improvements, theme customization, and admin tools  
**Changelog**: 0055, 0056, 0057, 0058, 0059, 0060, 0061

## Overview

Paideia LMS v0.7.1 brings significant improvements to the grading workflow, discussion forums, and administrative tools. This release extends grading capabilities to all module types, enhances discussion participation tracking, introduces theme customization options, and adds new admin monitoring tools. The submission management interface has been completely redesigned for better organization and usability.

## Major Features

### üìä Unified Grading System

**New Feature**: Grading is now available for all module types - assignments, quizzes, and discussions - with dedicated interfaces for each.

## Migration Guide

### Database Migration

This release includes a database migration that adds theme settings and updates quiz structure.

**Migration Steps**:
1. Upgrade to v0.7.1
2. The migration will be automatically applied when you start the server, or you can manually run:
   ```bash
   paideia migrate up
   ```
3. The migration will:
   - Add `color` and `radius` fields to `appearance_settings` table
   - Add `autoSubmitQuiz` task to job queue enums
   - Remove `time_limit` column from `quizzes` table (time limits now stored in quiz configuration)

**What the Migration Does**:
- Creates theme customization fields in appearance settings
- Updates job queue system to support auto-submit quiz tasks
- Removes deprecated `time_limit` field (replaced by `globalTimer` in quiz configuration)
- All changes are backward compatible

**After Migration**:
- ‚úÖ Server starts successfully
- ‚úÖ All existing functionality continues to work
- ‚úÖ Theme settings default to blue color and small radius
- ‚úÖ Quiz time limits continue to work (now using configuration instead of database field)
- ‚úÖ Auto-submit quiz jobs are properly tracked
- ‚úÖ All new features are immediately available

## Upgrade Instructions

### From v0.7.0 or Earlier

1. **Check migration status** (optional):
   ```bash
   paideia migrate status
   ```

2. **Upgrade to v0.7.1**:
   ```bash
   git pull origin main  # or your branch
   ```

3. **Start the server** (migration will be applied automatically):
   ```bash
   paideia start
   ```
   
   Or manually apply the migration first:
   ```bash
   paideia migrate up
   paideia start
   ```

4. **Configure Theme** (optional):
   - Navigate to Admin ‚Üí Appearance ‚Üí Theme
   - Select your preferred primary color
   - Adjust border radius if desired
   - Save changes

5. **Explore New Features**:
   - Visit Admin ‚Üí Server ‚Üí Scheduled Tasks to monitor background jobs
   - Check out the new discussion submission table in course modules
   - Try grading quiz submissions from the submissions table
