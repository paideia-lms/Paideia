# Release Notes - Paideia LMS v0.7.6

**Release Date:** December 25, 2025  
**Version:** 0.7.6  
**Type**: Bug Fix  
**Impact**: Medium - Improves reliability of sandbox mode database resets

## Overview

Paideia LMS v0.7.6 fixes a critical bug in the sandbox reset functionality that could lead to inconsistent database states. This release ensures that all database operations during sandbox resets are properly transactional, preventing partial data states if any operation fails.

**Issue**: Sandbox database resets could leave the database in an inconsistent state if the seed operation failed partway through.

**What Was Fixed**:
- **Transaction Consistency**: Seed operations now run within the same transaction as data deletion
- **Atomic Operations**: All sandbox reset operations (deletion + seeding) are now atomic
- **Error Recovery**: If any step fails, the entire operation is rolled back, maintaining database integrity

- **For Developers & Testers**:

  - No risk of partial database states after failed resets
  - Automatic rollback ensures database integrity is maintained
  - More reliable testing and development environment

**Technical Details**:
- Fixed missing transaction context in seed operation during sandbox reset
